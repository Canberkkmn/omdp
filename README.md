# OMDb movies
Project is getting movies from OMDb and shows them in a data table. Also you can click to see details page for that movie.

## Tech Stack
- ReactJS
- TypeScript
- Redux
- React Router
- Material UI
- Scss


## Installation
cd into the project directory and run:

````
cd omdp

yarn

yarn dev
````
